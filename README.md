# qvnari
いろいろな文字体系における“なりたち”絵文字のまとめ。

# 説明 - Description
いろいろな文字体系における“なりたち”というルーツを絵文字で取り上げます。</br>例えば漢字の字源やタイ・ラオ文字の字母名などです。
